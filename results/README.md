# Results — attribution

The arrays and figures in this directory are *derived scientific
results*, not software. They are released under the same MIT terms as
the code (see top-level LICENSE), but the appropriate form of reuse is
**citation**, not license-compliance: if you use these results, the
landscape data, or the figures (e.g. reproduced in a talk or paper),
please cite the accompanying preprint and this repository
(see ../CITATION.cff).

Contents:
- `nsx_x5_real.npz`  — real-axis residual landscape r(λ), λ ∈ [−0.10, 0.30]
- `nsx_x5_grid.npz`  — complex-window landscape r(λ) over the surveyed grid
- `figures/`         — landscape figures used in the paper

All are regenerable from `src/` via `docs/REPRODUCE.md`.
